### HOW TO PLAY:  
**move:** 'left arrow', 'right arrow'  
**jump and double jump:** 'up arrow'  
**attack level 1:** 'Z'  
**attack level 2+:** 'X'  
**items pickup:** 'C'  
**dialog:** 'space bar'
